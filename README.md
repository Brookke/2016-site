# 2016-site #

## How to install ##
### Development ###
#### Requirements ####
You will need the following things installed:
 - [NodeJs](https://nodejs.org/en/ "NodeJs")
 - [Go](https://golang.org/ "Go")

```bash
$ git clone 
$ cd 2016-site/
$ npm install -g grunt-cli bower
$ npm install
$ bower install
$ go get 
$ grunt build
$ go run main.go
```
